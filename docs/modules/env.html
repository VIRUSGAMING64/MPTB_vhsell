Resumen

Archivo pensado para extraer variables desde un fichero .env. Actualmente contiene únicamente un docstring y no implementa lógica.

Recomendaciones de implementación

  • Usar la librería python-dotenv o implementar un parser sencillo para cargar variables de entorno en el inicio de la aplicación.
  • Proveer una función load_env(path=".env") que lea el fichero y haga os.environ[KEY] = value si corresponde.
  • Asegurar que las variables sensibles no se cometen al repositorio y añadir .env a .gitignore.
