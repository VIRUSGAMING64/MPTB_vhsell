Resumen

Actualmente modules/database.py solo importa sqlite3. El módulo está marcado como punto de partida para la persistencia (SQLite), pero no contiene funciones implementadas aún.

Sugerencias de diseño

  • Crear una clase Database que encapsule operaciones básicas (conectar, ejecutar consultas, cierre seguro).
  • Usar migraciones simples (scripts SQL) o almacenar esquema en un archivo schema.sql.
  • Persistir información relevante: usuarios, metadatos de archivos subidos/descargados, logs de conversaciones si se requiere historial.
  • Implementar bloqueo/serialización de acceso si la base será usada desde múltiples hilos.
