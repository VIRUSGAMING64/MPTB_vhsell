Resumen

Define constantes de bitmask para clasificar chats, tipos de media y acciones. Usadas en varias partes del proyecto (p. ej. GetMedia en modules/utils.py y la cola de mensajes).

Constantes

Algunas constantes relevantes:

  • BANNED, GROUP, CHANNEL, MESSAGE, ADMIN, INVALID, MEDIA, ALERT — flags para chats.
  • DOCUMENT, VIDEO, AUDIO, PHOTO, VOICE, URL, OTHER, ALL, ALLOWED — flags para tipos de media.
  • DOWNLOAD, UPLOAD, DELETE — acciones.

Uso

Estas banderas se pueden combinar usando operaciones bit a bit (OR) y comprobar con AND para determinar presencia de tipos o permisos. Ejemplo conceptual:

if media_type & VIDEO:
    # es video
