Requisitos Previos

Para ejecutar este bot, necesitas tener instalado Python 3.8+ y las dependencias listadas en el proyecto.

Configuración

El bot utiliza variables de entorno para su configuración. Debes configurar las siguientes variables (puedes usar un archivo .env o exportarlas en tu terminal):

TOKEN=tu_token_de_telegram
API_HASH=tu_api_hash_de_telegram
API_ID=tu_api_id_de_telegram
OPEN_AI=tu_api_key_de_openai

Ejecución

Para iniciar el bot, ejecuta el archivo principal desde la raíz del proyecto:

python bot.py

El bot iniciará el polling y comenzará a escuchar mensajes. Verás un mensaje en la consola indicando que el bot está corriendo.
